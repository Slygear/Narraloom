# Narraloom

Narraloom is an open-source **visual novel engine and editor** focused on
narrative-driven games, branching stories, and rich UI.

The goal of Narraloom is to provide a modern, flexible alternative to classic
visual novel engines while remaining simple enough for solo developers.
